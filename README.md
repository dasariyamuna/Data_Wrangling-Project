# Data Wrangling Project

This project demonstrates how to clean raw CSV data using Python and Pandas.

## Steps Performed
- Loaded raw dataset
- Identified missing values and duplicates
- Cleaned data using Pandas
- Generated analysis-ready dataset

## Files
- data/raw/dataset.csv – original data
- data/cleaned/cleaned_dataset.csv – cleaned data
- scripts/clean_transform.py – cleaning logic
- DATA_DICTIONARY.txt – column explanations

## Tools Used
Python, Pandas
